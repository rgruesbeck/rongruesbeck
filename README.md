# rongruesbeck.com
http://rgruesbeck.github.io/rongruesbeck/

## Install
```sh
npm install
```

## Dev Servers
To serve src files:
```sh
npm run srcserver
```
To serve build files:
```sh
npm run buildserver
```

## Build
```sh
npm run build
```

## Deploy
```sh
npm run build
git add .
git commit
git push origin master
npm run deploy
```
