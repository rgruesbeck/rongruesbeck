# rongruesbeck.com
http://rgruesbeck.github.io/rongruesbeck/

## Install
```sh
npm install
```

## Get Started
Start build script
```sh
npm start
```
Open a new shell and start the server
```sh
npm run server
```

## Deploy
```sh
npm run deploy
```
